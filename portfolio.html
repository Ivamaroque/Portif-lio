Projeto ConSec

Metodologia de Design

A estrutura deste projeto foi construída utilizando Auto Layout no Figma. Essa abordagem garante que todos os componentes sejam flexíveis e responsivos, facilitando futuras edições e a manutenção da consistência visual em todo o sistema, sem a necessidade de ajustes manuais de posicionamento.

Projeto Diário Oficial do MPRN

Metodologia e Features

Toda a interface foi desenvolvida com Auto Layout para garantir total responsividade e facilidade de manutenção. Pensando na experiência do usuário, o projeto inclui variantes para Light e Dark Mode, além de uma versão mobile dedicada, assegurando usabilidade e conforto visual em qualquer dispositivo ou condição de iluminação.
